# 公式索引

每条公式标注可算性：
- **可算** = 给定输入可以直接算出数值
- **需定义量表** = 结构成立，但子项需要先定义评分标准
- **待校准** = 概念占位，权重和子项都未定

## 2. 动态熵

### Shannon 熵 【可算】

```
H(X) = -Σ p(x) log p(x)
```

### 当前轮认知熵 【可算（若 p_t 已知）】

```
H_t = -Σ p_t(ω) log p_t(ω)
```

### 路径权重 【需定义量表】

```
q_t(ω) = Rel(ω) · Ev(ω) · Pri(ω) · Ctx(ω)
p_t(ω) = q_t(ω) / Σ q_t(ω)
```

子项 Rel/Ev/Pri/Ctx 各需要评分标准。

### 运行熵效应 【可算（若 H_before 和 H_after 可观测）】

```
ΔH_run(n,t) = H_after(n,t) - H_before(t)
```

### 降熵贡献 【可算（若 H_t 可观测）】

```
ER(n,t) = H_t - H_t^(n)
ER_avg(n) = (1/N_round) Σ_t ER(n,t)
```

### 目标熵偏差 【需定义量表】

```
ΔH_l = H_l_current - H*_l
```

`H*_l` 需要针对任务预设。

## 4. 共同属性空间

### 交会度 【可算（若属性子集已标注）】

```
I(X, Y) = |A ∩ B| / |A ∪ B|
```

### 差异度 【可算（若属性子集已标注）】

```
Diff(X | Y) = |A - B| / |A|
```

### 三层交会 【可算】

```
I(E_i, K_j, T_s) = |A_i ∩ B_j ∩ C_s| / |A_i ∪ B_j ∪ C_s|
```

### 四层交会 【可算】

```
I(E_i, K_j, T_s, P_h) = |A_i ∩ B_j ∩ C_s ∩ D_h| / |A_i ∪ B_j ∪ C_s ∪ D_h|

```

## 5. Möbius 反演

### 一般式 【可算（若 F(S) 可观测）】

```
G(S) = Σ_{A⊆S} (-1)^{|S|-|A|} F(A)
```

### 展开式

```
单层:  G(E) = F(E) - F(∅)
两层:  G(EK) = F(EK) - F(E) - F(K) + F(∅)
三层:  G(EKT) = F(EKT) - F(EK) - F(ET) - F(KT) + F(E) + F(K) + F(T) - F(∅)
四层:  G(EKTP) = F(EKTP) - ΣF(3层) + ΣF(2层) - ΣF(1层) + F(∅)
```

### 总效果函数 【待校准】

```
F(S) = θ1·Q(S) + θ2·C(S) + θ3·V(S) + θ4·L(S)
```

Q=输出质量, C=覆盖, V=有效性, L=闭环表现。子项量化方式待定。

## 7. 评分体系

```
S_struct = αS_cover + βS_diff + γS_pair + δS_tri + εS_quad
```

```
S_cover = |A ∪ B ∪ C ∪ D| / |U|
```

```
S_diff = 1 - |Diff_actual - Diff_target|
```

### 层间接口偏差分 【需定义量表】

```

```

### 总分 【可算（若子分已有）】

Score_total = λS_struct + μS_entropy + νS_valid + ξS_revise

### 审核目标函数 【可算（若子分已有）】

## 可算性统计

| 状态 | 数量 | 说明 |
|------|------|------|
| **可算** | ~10 条 | 主要是集合运算（交会/差异/覆盖）和反演展开式 |
| **需定义量表** | ~20 条 | 结构成立，但子项需要"属性值→数值"的映射表 |
| **待校准** | ~5 条 | F(S) 的具体实现、所有权重系数 |

**结论**：整套体系中，真正能直接算的只有集合运算部分。其余公式要落地，核心瓶颈是"量表定义"——即每个子项到底怎么从标签映射成数字。计算指南（04）将解决这个问题。
