# 符号速查表

## 四层总集合

| 符号 | 含义 |
|------|------|
| `E = {e_1, ..., e_m}` | 经验层总集合 |
| `K = {k_1, ..., k_n}` | 知识层总集合 |
| `T = {t_1, ..., t_p}` | 思维层总集合 |
| `P = {p_1, ..., p_q}` | 哲学层总集合 |
| `V = E ∪ K ∪ T ∪ P` | 全体节点集合 |
| `R ⊆ V × V` | 边集合（注意不用 E 表示边，避免与经验层冲突） |

## 经验层节点属性

| 字段 | 含义 | 典型取值 |
|------|------|----------|
| `content_i` | 经验内容 | 自由文本 |
| `source_i` | 来源类型 | personal / observed / reported / derived |
| `mediation_i` | 中介程度 | direct / mediated |
| `process_i` | 加工层级 | raw / organized / reflective |
| `validity_i` | 有效性 | unverified / supported / contested / stable |
| `function_i` | 结构功能 | evidence / counterexample / anomaly / boundary |

## 知识层节点属性

| 字段 | 含义 | 典型取值 |
|------|------|----------|
| `content_j` | 知识内容 | 自由文本 |
| `scope_j` | 适用范围 | personal / domain / general |
| `abstraction_j` | 抽象层级 | case / rule / principle |
| `justification_j` | 成立方式 | evidential / inferential / systematic |
| `function_j` | 结构功能 | descriptive / normative / procedural |
| `stability_j` | 稳定性 | tentative / supported / stable / contested |

## 思维层五级梯度

| 符号 | 名称 | 核心任务 | 克服上一级的 |
|------|------|----------|-------------|
| `T_repr` | 表象思维 | 抓取印象、联接 | — |
| `T_det` | 定型思维 | 对象化、分类、边界 | 表象的漂浮性 |
| `T_disc` | 推演思维 | 规则链、条件链、论证 | 定型的静止性 |
| `T_dial` | 辩证思维 | 反例、矛盾、边界修正 | 推演的框架封闭性 |
| `T_arch` | 统摄思维 | 主线、整体组织、重排 | 辩证的纯否定倾向 |

递进：`T_repr → T_det → T_disc → T_dial → T_arch`
回落：`T_arch → T_disc → T_det → E/K`

## 哲学层四段结构

| 符号 | 名称 | 作用 |
|------|------|------|
| `P0` | 入口清障 | 去假象/偏见/旧体系误导（培根四假象） |
| `PΔ` | 差异层 | 定义哪些差异合法、必须保留 |
| `PI` | 同一层 | 保证稳定性、非矛盾、可追溯 |
| `PΣ` | 综合统一 | 差异与同一的更高组织 |

## 共同属性空间与映射

| 符号 | 含义 |
|------|------|
| `U = {u_1, ..., u_m}` | 共同属性空间（四层可比较的公共平面） |
| `P(U)` | U 的幂集 |
| `φ_E : E → P(U)` | 经验节点映射到属性子集 |
| `φ_K : K → P(U)` | 知识节点映射到属性子集 |
| `φ_T : T → P(U)` | 思维节点映射到属性子集 |
| `φ_P : P → P(U)` | 哲学节点映射到属性子集 |
| `A_i = φ_E(e_i)` | 经验节点的属性子集 |
| `B_j = φ_K(k_j)` | 知识节点的属性子集 |
| `C_s = φ_T(t_s)` | 思维节点的属性子集 |
| `D_h = φ_P(p_h)` | 哲学节点的属性子集 |

## 交会与差异

| 符号 | 含义 |
|------|------|
| `I(X, Y) = \|A∩B\| / \|A∪B\|` | Jaccard 交会度 |
| `Diff(X \| Y) = \|A-B\| / \|A\|` | 相对差异度 |

## 动态熵相关

| 符号 | 含义 | 性质 |
|------|------|------|
| `H_pos(n)` | 节点层位熵（静态位置） | 结构量 |
| `H_t` | 当前轮认知熵 | 运行量 |
| `ΔH_run(n,t)` | 节点在时刻 t 的运行熵效应 | >0 发散, <0 收束 |
| `ER(n,t) = H_t - H_t^(n)` | 节点降熵贡献 | >0 收束, <0 发散 |
| `ER_avg(n)` | 平均降熵贡献 | 跨轮统计 |
| `H* = [H*_E, H*_K, H*_T, H*_P]` | 目标熵向量 | 各层合理运行区间 |
| `ΔH_l = H_l_current - H*_l` | 当前偏差 | >0 过散, <0 过紧 |
| `Ω_t = {ω_1, ..., ω_m}` | 候选路径集合 | — |

## 跨层接口函数

| 符号 | 方向 | 含义 |
|------|------|------|
| `Build_K(E', k_j)` | E → K | 经验子集对知识节点的生成能力 |
| `K_potential(e_i)` | E → K | 单经验节点的知识化潜力 |
| `Act_ET(e_i, t_s)` | E → T | 经验对思维的激活值 |
| `Act_KT(k_j, t_s)` | K → T | 知识对思维的激活值 |
| `GateFit_EP(e_i, g)` | P → E | 经验节点与哲学门控的适配度 |
| `GateFit_KP(k_j, g)` | P → K | 知识节点与哲学门控的适配度 |
| `Act(n,t)` | 调度 | 节点优先激活值 |

## 反演相关

| 符号 | 含义 |
|------|------|
| `F(S)` | 层子集 S 的总效果 |
| `G(S)` | 层子集 S 的纯贡献（剥离低阶重复） |
| `G(S) = Σ_{A⊆S} (-1)^{\|S\|-\|A\|} F(A)` | Möbius 反演一般式 |
| `W_E, W_K, ...` | 单层权重 = G(单层) |
| `W_EK, W_ET, ...` | 两层权重 = G(两层) |
| `W_EKT, ...` | 三层权重 = G(三层) |
| `W_EKTP` | 四层权重 = G(四层) |

## 八个结构量

| 符号 | 层级 | 一句话 |
|------|------|--------|
| `Core(n)` | 节点层 | 节点是否真正主干 |
| `Couple(u,v)` | 边层 | 两节点是否形成稳定有效关系 |
| `Mod(M_i)` | 局部结构 | 子结构是否真正成模块 |
| `Red(S)` | 局部结构 | 是否存在结构冗余 |
| `Conflict(S)` | 局部结构 | 是否存在互斥冲突（≠合法差异） |
| `Closure(S)` | 整体治理 | 是否形成闭环 |
| `Writeback(S)` | 整体治理 | 运行结果能否沉淀回结构 |
| `Gate(g)` | 整体治理 | 门控规则是否真正起作用 |

## 评分体系

| 符号 | 含义 |
|------|------|
| `S_struct` | 结构总分 = αS_cover + βS_diff + γS_pair + δS_tri + εS_quad |
| `S_cover` | 覆盖分 = \|A∪B∪C∪D\| / \|U\| |
| `S_diff` | 差异分 = 1 - \|Diff_actual - Diff_target\| |
| `S_pair` | 两层耦合分 |
| `S_tri` | 三层协同分 |
| `S_quad` | 四层协同分 |
| `S_edge` | 层间接口偏差分 |
| `Score_static` | 静态审核分（设计质量） |
| `Score_runtime` | 动态审核分（运行质量） |
| `Score_total` | 总联合评分 |
| `J` | 输出前审核目标函数（决定放行/重排） |

## 通用约定

- `α, β, γ, δ, ε, ...` — 可调权重（非负，需校准）
- `N_round` — 总轮数（不用 T，避免与思维层冲突）
- `t` — 当前时刻/轮次
- `∪ ∩ - ⊆ ∅ × ∈` — 标准集合运算
